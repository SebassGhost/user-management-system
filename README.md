# User Management System (Python)

## Description

The User Management System is a simple command-line application developed in Python. It allows basic user administration through a terminal interface, focusing on clean logic, data validation, and file persistence. This project was created as a foundational portfolio piece to demonstrate problem-solving skills and good coding practices.

## Features

* Register new users
* List all registered users
* Search users by ID or email
* Delete users by ID
* Persistent storage using a JSON file

## Technologies Used

* Python 3
* JSON for data storage
* Command Line Interface (CLI)

## Project Structure

* data/: stores user data in JSON format
* src/: contains the main Python application logic
* README.md: project documentation

## How to Run

1. Make sure Python 3 is installed on your system
2. Clone this repository
3. Navigate to the project folder
4. Run the application using:
   python src/main.py
